# Requirements

Make sure you read this section thoroughly before proceeding with any installation of Overleaf.

### Requirements for installing Server Pro and Community Edition

{% hint style="warning" %}
We strongly recommend you run Server Pro on a recent version of Ubuntu with a recent version of Docker. Running Server Pro on other systems is possible, but will make it much harder for us to offer support for non-default Ubuntu/Docker setups. We also cannot offer support for problems with Docker itself.
{% endhint %}
